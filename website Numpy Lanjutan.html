Operasi Matriks NumPy

NumPy menyediakan operasi matriks seperti penjumlahan, pengurangan, dan perkalian.

import numpy as np

A = np.array([[1, 2], [3, 4]])
B = np.array([[5, 6], [7, 8]])
C = A + B  # Penjumlahan
print(C)

D = np.dot(A, B)  # Perkalian matriks
print(D)

Operasi Set NumPy

Melakukan operasi himpunan seperti union, intersection, dan difference.

a = np.array([1, 2, 3, 4])
b = np.array([3, 4, 5, 6])
print(np.union1d(a, b))       # Gabungan
print(np.intersect1d(a, b))   # Irisan
print(np.setdiff1d(a, b))     # Selisih

Vektorisasi NumPy

Melakukan operasi langsung antar elemen array tanpa loop eksplisit.

a = np.array([1, 2, 3])
b = np.array([4, 5, 6])
print(a + b)      # Penjumlahan
print(a ** 2)     # Pangkat dua

Boolean Indexing

Memfilter elemen array berdasarkan kondisi.

a = np.array([1, 2, 3, 4, 5])
print(a[a > 3])  # Hanya elemen > 3

Fancy Indexing

Mengakses banyak elemen dengan daftar indeks.

a = np.array([10, 20, 30, 40, 50])
indices = [1, 3, 4]
print(a[indices])

Broadcasting

Memungkinkan operasi antar array dengan bentuk berbeda.

a = np.array([1, 2, 3])
b = np.array([[10], [20], [30]])
print(a + b)
